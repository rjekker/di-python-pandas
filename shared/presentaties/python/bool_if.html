Booleans

A boolean expression can either be true or false.

Predefined values: True and False 

>>> 5 > 2
True

Boolean operators

x == y  # x is equal to y
x != y  # x is not equal to y
x > y   # x is greater than y
x < y   # x is less than y
x >= y  # x is greater than or equal to y
x <= y  # x is less than or equal to y
x in y  # x is an element of y (string or list)

Note: == (comparison) vs. = (assignment)!

Logical operators

Combine boolean expressions using logical operators and, or, not: 

weekend = (day == "sat") or (day == "sun")
is_fish = has_fins and not is_mammal

Conditional Execution

The if-statement runs code depending on the value of a boolean expression: 

if bank_balance > 0:
    print('We have some money!')
else:
    print('We are broke :(')

The else part is optional.

Nested if statements

age = int(input("Age? "))
gender = input("Gender (m/f)? ")

if age >= 18:
    if gender == "m":
        print("Hello sir")
    else:
        print("Hello madam")
else:
    if gender == "m":
        print("Hey boy")
    else:
        print("Hey girl")

Nesting

This can get out of hand.. 

language = input("What language do you speak? ")

if language == "Dutch":
    print("Hoi")
else:
    if language == "English":
        print("Hello")
    else:
        if language == "French":
            print("Bonjour")
        else:
            if language == "German":
                print("Gutentag")
            else: # etc. etc.

Elif

Use elif to keep the code clean: 

language = input("What language do you speak? ")

if language == "Dutch":
    print("Hoi")
elif language == "English":
    print("Hello")
elif language == "French":
    print("Bonjour")
elif language == "German":
    print("Gutentag")
else:
    print("I don't speak that one :(")

Wat is truth?

Whenever you test for truth (e.g. an if-statement), the following values will be seen as False:

  • Constants None en False.
  • The number zero: 0, 0.0, (etc)
  • Empty collections: '', (), [], {}, set(), range(0)

False: zero

sales = beers_sold + wine_sold + soda_sold + coffee_sold
if not sales: # if sales == 0
    print("Nothing sold today :(")
else:
    # etc.

False: Empty string

if not mail.subject: # mail.subject == ""
    print("Send  mail with an empty subject?")
