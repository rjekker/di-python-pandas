Tuples

Tuples

  • Sequence with zero or more values
  • Very similar to lists
  • Support most list operations (indices, for, len, max, min, etc.)
  • Difference: tuples are immutable

Tuples

A tuple is a list of values separated by comma's 

t = 'a', 5, [1, 2]
t2 = ('a', 5, [1,2]) # optional parens
t == t2 # True
type(t) # type 'tuple'
t = ()  # empty tuple

Tuple with 1 element

Watch the comma! 

t = (5)
type(t) # type 'int' !!!
t = (5,) # tuple!

Assignment

Cannot assign to a tuple element: 

t = (12, 'augustus', 2019)
t[1] = 'september'

TypeError: 'tuple' object does not support item assignment

Append

Cannot add elements: 

t = ()
t.append('a')

AttributeError: 'tuple' object has no attribute 'append'

Using tuples

Lists where the position of a value is significant 

coordinates = (3, 8)  # x and y axes
address = ("My street", "74 a", "Amsterdam", "The Netherlands")
date = (2020, 7, 14)
  • append or insert don't make sense here
  • compare with dict or objects

As return values

Function with more than one return value 

def min_max(l):
    return min(l), max(l)

t = min_max([5,8,3,1,9])

Tuple unpacking

Assign to multiple variables at once 

t = (13, 'EUR')
amount, currency = t

dict items

When looping over a dict; return both key and value 

capitals = { 'DE': 'Berlin', 'FR': 'Paris', 'NL': 'Amsterdam' }
for k, v in capitals.items():
    print(v, "is the capital of", k)

Berlin is the capital of DE
Paris is the capital of FR
Amsterdam is the capital of NL

Enumerate

When looping over a list; return both index and value 

for index, value in enumerate(['a', 'b', 'c']):
         print(index, ':', value)

0 : a
1 : b
2 : c
