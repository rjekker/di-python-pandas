Address book

Consider the following code: 

addresses = {
    "Pluralsight": "182 N. Union Ave, Farmington, UT 84025",
    "Apple": "One Infinite Loop. Cupertino, CA 95014",
    "Microsoft": "One Microsoft Way, Redmond, WA 98052-6399"
}

contact = input("Contact name? ")
print(contact, ":\n------------")
print(addresses[contact])

Please add exception handling to detect unknown contacts.

Open a file

filename = input("Enter a file to open? ")
f = open(filename)

This code can cause a number of different exception types. Try to find at least 3 different types of exceptions and write code to handle each type with its own error message.

Safely enter a number

Write a function input_number() that asks the user to input a number.

If the input is not a number, you repeat the question. Only when the input is a valid number, you return the value.

So there should be a loop inside the function that does not end until the user has entered a valid number.

Bonus: Create a custom exception and throw it

Create a function check_payment_type(s) that checks the value of s.

Valid values for s are "cash" and "creditcard". If s is anything else, throw a InvalidPaymentTypeException.

You will need to do some research to learn how to create your own exception type.
