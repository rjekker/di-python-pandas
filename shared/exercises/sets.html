A set from a string

Consider the following code: 

w = input("A word please? ")
print(set(w))

Run this code and enter a (long) word. What does it print?

Change the code so that it prints the number of unique characters in the word.

Your own set

Create a set with a bunch of random, large numbers you choose yourself.

Can you print these numbers sorted by size?

Can you print the maximum and minimum number in the set?

What about the sum of the numbers in the set?

Numbers

Consider the following two lists of numbers: 

# This generates lists of numbers using a "list comprehension"
import math
numbers1 = [math.floor(math.sqrt(x*y)) for x in range(0, 100, 3) for y in range(0,100,7)]
numbers2 = [math.floor(math.sqrt(x*y)) for x in range(0, 100, 8) for y in range(0,100,19)]

Use a set to determine:

  • how many unique numbers are in the first list.
  • how many unique numbers are in the second list.
  • how many numbers are in the first list only
  • how many numbers are in the second list only
