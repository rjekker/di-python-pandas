Name

Ask the user for their name and print:

  • a friendly greeting
  • the length of their name

Example output 

What is your name? Reindert

Hi, Reindert!
Your name is 8 characters long.

Sentence

Write a program that asks that user to enter an article, a noun, and a verb. The program then creates a sentence of the form article noun verb.

Example output: 

Enter an article? The
Enter an noun? dog
Enter a verb? barks

Sentence: "The dog barks."

Convert mile to kilometer

Write a program that reads a mile in a double value from the console, converts it to kilometers, and displays the result. The formula for the conversion is as follows: 1 mile = 1.6 kilometers

Here is a sample run: 

Enter miles: 96
96 miles is 153.6 kilometers

Temperatures

Write a program that asks the user to enter a temperature in degrees Celsius. The program will convert the temperature to degrees Fahrenheit and display the result to the user with an appropriate message. (Hint: F° = (1.8 × C°) + 32.)

Example output: 

Enter a temperature in Celsius? 40

That is 104.0 degrees Fahrenheit.

Speed

A cyclist wants to keep track of his performance after exercising.

Write a program that:

  • asks for the distance he covered (in miles)
  • asks for the time he needed (in minutes)

Then, display the average speed in miles per hour.

Example output: 

Distance (miles)? 10
Time (minutes)? 6

Your average speed was 100.0 miles per hour

Boxes

Write a program that prompts the user to enter the length (L), width (W), and height (H) of a box. The program will then compute and display the volume and surface area of the box. (Hint: Volume = L × W × H; Surface area = 2×[(L × W) + (L × H) + (H × W)].)

Example output: 

length? 10
width? 10
height? 10

The volume is 1000 and the surface is 600.

Digits

Write a program that reads an integer between 0 and 1000 and multiplies all the digits in the integer. For example, if an integer is 932 , the multiplication of all its digits is 54.

Hint: Use the % operator to extract digits, and use the / operator to remove the extracted digit. For instance, 932 % 10 = 2 and 932 / 10 = 93.

Here is a sample run: 

Enter a number between 0 and 1000: 999
The multiplication of all digits in 999 is 729

Triangles

Write a program that reads the lengths of the two sides of a right triangle and calculates the length of the third side. Hint: if the first side is a and the second side is b, the third side c is the square root of (a x a) + (b x b).

Note: I did not teach you how to calculate a square root in Python.. Try to search for that yourself!

Example output: 

First side: 3
Second side: 4

The length of the third side of this triangle is 5.0
