Input and Output

  • Use input() to ask the user for input
  • Use print() to show one or more values.
name = input("What is your name? ")
print("Hi ", name, "!")

Result:
What is your name? Reindert
Hi Reindert !

Print and scripts

Why do we need print in a script but not in a console?

Operating on Input

order = input("A bread is $2. How many do you need? ")
print("That will be $", 2*order)

What will go wrong here?

input() always returns a string!!

Conversion to int

order = input("A bread is $2. How many do you need? ")
print("That will be $", 2*int(order))

Another problem

birth_day = 15
print("My birthday is on april " + birth_day)

What will go wrong here?

You cannot add a string and an int

Conversion to string

birth_day = 15
print("My birthday is on april " + str(birth_day))

Another solution

Print multiple values of any type: 

birth_day = 15
print("My birthday is on april", birth_day, "and you are invited!")

Drawback: print() adds spaces between values.

Third solution: F-strings

Put an f in front of a string and use your variables inside the string: 

birth_day = 15
print(f"My birthday is on april {birth_day} and you are invited!")

My birthday is on april 15 and you are invited!

Type Conversions

There are functions for type conversions for all builtin types: 

 >>> str(5.8)
'5.8'
 >>> int('10')
 10
 >>> int(5.6)
 5
 >>> float(5)
 5.0
 >>> float('5')
 5.0
 >>> list("hello")
 ['h', 'e', 'l', 'l', 'o']
