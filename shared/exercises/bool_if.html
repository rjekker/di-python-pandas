Pass or fail

Ask the user to enter a test score (a number between 0 and 100). If it is 60 or higher, print "Pass", else print "Fail" 

Your score? 55
Fail

Your score? 70
Pass

Tax brackets

Ask the user to input his income in dollars. Then print how much taxes he has to pay according to the following (imaginary) rules:

  • Anything below $ 20,000: no taxes
  • $20,000 - $50,000: 30%
  • Everything above: 60%

Example: 

How much do you earn? 10000
You don't have to pay taxes!

How much do you earn? 100000
You have to pay 60%

Largest

Write a program that asks the user to enter three numbers. The program will then determine and print the largest of the three numbers.

Example: 

first number? 2
second number? 7
third number? 4

Largest: 7

Vowels

Write a program that asks the user to enter a letter and check whether it is a vowel or a consonant. If it is neither, you show an error message.

Sample: 

Enter a letter: x
x is a consonant

Enter a letter: 5
5 is neither a vowel or a consonent.

Multiplication Quiz

Write a program that prints two random integers. The user has to enter the product of these two integers. Check that the answer was correct and show this to the user.

An example run of the program: 

What is 12 x 12? 144
That is correct!

What is 13 x 12? 153
That is wrong :(

Hint: Use random.randint() to generate random numbers as follows: 

import random
x = random.randint(0,15) # A random number between 0 and 15
print(x)

Cost of shipping

A shipping company uses the following function to calculate the cost (in dollars) of shipping based on the weight of the package (in pounds).

Cost for weight w is:

  • 2.5, if 0 < w <= 2
  • 4.5, if 2 < w <= 4
  • 7.5, if 4 < w <= 10
  • 10.5, if 10 < w <= 20

Write a program that prompts the user to enter the weight of the package and display the shipping cost. If the weight is greater than 20, display a message “the package cannot be shipped".

Sample run: 

Weight? 6
Cost:  7.5
