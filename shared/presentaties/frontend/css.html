Selectors

CSS code consists of a selector and applied rules: 

p {
    color: red;
}

Don't forget the semicolon!

Selecting by class

<p class="intro">Hi! I'm RJ</p>
<p class="other">Something else</p>

p { color: black; }
.intro { font-weight: bold }
p.other { margin-left: 50px; }
