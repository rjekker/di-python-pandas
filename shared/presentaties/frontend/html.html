Frontend/Backend

The frontend is concerned with presenting info to the user.

Think: images, colors, animations, forms.

The backend is concerned with logic and data storage.

HTML

HyperText Markup Language

HTML

HTML gives structure to a document:

  • paragraphs
  • sections
  • title
  • links
  • headers

Usually HTML documents are viewed in a browser

Minimal document

<!DOCTYPE html>
<html>
<head>
    <title>Page title here</title>
</head>
<body>
    This text will go on page
</body>
</html>

HTML elements

Are represented by tags 

<h1>This is a header</h1>
<p>This is a paragraph</p>
<p>A part of this paragraph
  <em>will be emphasized</em>
but the rest is not.</p>

Closing tags?

Most tags can contain text or other tags and need to be closed: <h2>...</h2>

In some cases the closing tag is optional or not allowed:

  • <p>
  • <br>
  • <hr>
  • <img>
  • <input>
  • <meta>
