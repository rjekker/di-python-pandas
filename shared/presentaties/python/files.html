os

os is een module voor os-gerelateerde functies. Onder andere bevat deze functies voor file-manipulatie.

  • OS documentatie
  • File en Directory functies

Voorbeelden

  • os.chdir(path)
  • os.getcwd()
  • os.listdir()
  • os.mkdir()
  • os.remove(), os.rmdir()

os.path

os.path is een module waarmee je bestandsnamen kunt manipuleren

Werkt volledig cross-platform

Belangrijke functies

  • os.path.join() voor het samenstellen van een pad
  • os.path.exists(), ..isfile(), ..isdir()

Zie https://docs.python.org/3/library/os.path.html

Een pad samenstellen

import os, os.path
# een relatief pad
pad = os.path.join('mijndir' )

# Een absoluut pad
# Gebruik os.sep: dit is een slash
homedir = os.path.join(os.sep, 'home', 'reindert')

Files list

from os import listdir

homedir = os.path.join(os.sep, 'home', 'reindert')

files = listdir(homedir)
for f in files:
    if f.endswith("txt"):
        print(f)
