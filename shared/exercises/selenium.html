1. Wiki

Ask the user for a word. Print the first 3 paragraphs of text from the wikipedia article about that word.

Screenshot

Save a screenshot of the page using selenium.

Hint: see Browser Manipulation

Random

Select a random link on the page and click it. Print the title and the url of the page that you end up on.

2. Price

Visit https://codesensei.nl/web/prices

The prices on this page appear dynamically, after the page has loaded.

Waiting

Ask the user for a stock name. When a price for that stock is shown on the page, print it and exit.

Ticker

Add a loop that waits for the price to change and prints the new prices as they appear.

3. Login

(This is a re-implementation of a previous exercise with requests) Write a script that goes to https://codesensei.nl/accounts/login/ and login with the following:

username nomura
password python

You will be rewarded with a nice message that appears on the page after login. Scrape this from the page and print it.

Hint 1

The most challenging part of this is to find the form elements in the HTML.

The username and password input can be found by their name attribute.

For the submit button, use the Xpath expression '//input[@type="submit"]'

Hint 2

After clicking the submit button, there will be a page refresh, followed by a new page with a quote.

You have to use WebDriverWait to wait for the quote to show up, see Wait documentation

4. Download

Consider https://codesensei.nl/web/download. Write a script that:

  • Clicks the first link
  • Waits for the second link to appear
  • Clicks the second link
  • Downloads the resulting file
