Sets

Sets

  • Sequence with zero or more values
  • Elements are unique (no duplicates)
  • Supports sequence operations (indices, for, len, max, min, etc.)
  • Nice for deduplication and set theory operations

Defining a set

print({ 2, 4, 6, 8 })

{8, 2, 4, 6}
  • Note: elements are unordered!

Set with duplicates

s = { 2, 2, 4, 4, 4, 6, 6, 8, 8 }
print(f"{s} is a {type(s)} with length {len(s)}")

{8, 2, 4, 6} is a <class 'set'> with length 4
  • Nice for deduplication

Empty set

s = set()
print(f"{s} is a {type(s)} with length {len(s)}")  

set() is a <class 'set'> with length 0
  • Why can we not use {} here?

Element types

Elements must be immutable (int, str, bool, tuple, …) 

# This is fine:
print({"a", "b", "b", 2, 2, (44, 55), (44, 55)})
# This is not fine:
print({[1], []})
print({{}})  # A set containing a dict !?

{'b', 2, (44, 55), 'a'}  # Line 1 is ok
TypeError: unhashable type: 'list'
TypeError: unhashable type: 'dict'

Creating sets

From a string

s = set("Hello everyone!")
print(s)

{'o', 'v', 'y', 'n', 'l', 'r', 'e', '!', ' ', 'H'}

From a list

s = set([1, 2, 1, 2, 3, 4, 5, 4, 3, 2, 3, 2, 3, 4])
print(s)

{1, 2, 3, 4, 5}

Membership

Use in and not in: 

s = {"a", "b", "b", 2, 2, (44, 55), (44, 55)}
print((44,55) in s)
print("c" not in s)

Operators

Union

Return a set with all elements from both sets 

s1 = {'a', 'b', 'c'}
s2 = {'c', 'd', 'e'}
print( s1 | s2 )
# equivalent: s1.union(s2)

{'a', 'e', 'c', 'd', 'b'}
  • Note: 'c' only occurs once in the result! Why?

Intersection

Return a set with elements common to both sets 

s1 = {'a', 'b', 'c'}
s2 = {'c', 'd', 'e'}
print( s1 & s2 )
# equivalent: s1.intersection(s2)

{'c'}

Difference

Return a set with elements that are in the first set but not in the second 

s1 = {'a', 'b', 'c'}
s2 = {'c', 'd', 'e'}
print( s1 - s2 )
# equivalent: s1.difference(s2)

{'b', 'a'}
