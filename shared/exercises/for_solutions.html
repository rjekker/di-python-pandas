A list of words

Ask the user to enter 4 words. Put those in a list with append. Then try the following:

  • Sort the list and print it.
  • Can you find out how to sort the list in reverse order?
  • Change each word into uppercase before printing it
  • Print only the first letter of each word in the list
l = []
for i in range(4):
    l.append(input("A word? "))

print(sorted(l))
print(sorted(l, reverse=True))

for w in l:
    print(w.upper())
    print(w[0])

Tuition

Suppose that the tuition for a university is $10,000 this year and increases 6% every year. In one year, the tuition will be $10,600. Write a program that computes the tuition in ten years and the total cost of four years’ worth of tuition after the tenth year. 

tuition = 10000
increase_percentage = 6

for year in range(10):
    print("Tuition for year", year, "is", tuition)
    tuition += (tuition / 100 * increase_percentage)

print("---")
print("Tuition after ten years: ", round(tuition))

# Now let's calculate what four years of tuition
# will be now
total = 0
for year in range(4):
    total += tuition
    tuition += (tuition / 100 * increase_percentage)

print("Tuition for 4 years at this point will be", round(total))

Pyramid

Display numbers in a pyramid pattern) Write a nested for loop that prints the following output (don't worry too much about the whitespace): 

                                                        1
                                                1       3       1
                                        1       3       9       3       1
                                1       3       9       27      9       3       1
                        1       3       9       27      81      27      9       3       1
                1       3       9       27      81      243     81      27      9       3       1
        1       3       9       27      81      243     729     243     81      27      9       3       1
1       3       9       27      81      243     729     2187    729     243     81      27      9       3       1

def list_to_string(l):
    s = ""
    for num in l:
          s += str(num) + "\t"
    return s

# We start with only the number 1
l = [1]


for x in range(8):
    # We have to start with a certain number of tabs
    print((8-x-1)*"\t" + list_to_string(l))
    # create new line with numbers
    # by multiplying current line with 3
    # and adding a 1 at the front and the end
    new_l = [1]
    for n in l:
        new_l.append(n*3)
    new_l.append(1)
    l = new_l

Bonus: Store

We have a shop with 4 products and their prices: 

products = ["Cheese", "Bread", "Coffee", "Wine"]
prices = [8, 2, 4, 3]

Create a program that:

  • prints a list of products and asks the user to choose one
  • when the user makes a choice, he can enter an amount
  • calculate and print the total price for his order
products = ["Cheese", "Bread", "Coffee", "Wine"]
prices = [8, 2, 4, 3]
order = []

def print_products():
    for i in range(len(products)):
        print(i, ":", products[i])

def print_order():
    print("Your order:")
    print("-------------------------------------------")
    total_price = 0
    for line in order:
        choice = line[0]
        amount = line[1]
        product = products[choice]
        price = prices[choice]
        print(amount, product, ":", price*amount)
        total_price += price*amount
    print("------------------------------------------")
    print("Total price:", total_price)


while True:
    print_products()
    print("Enter number of your choice, or -1 to exit")
    print("------------------------------------------")
    choice = int(input("Choose product: "))
    if choice < 0:
        break
    if choice >= len(products):
        print("Not a valid choice, please try again")
    else:
        product = products[choice]
        print("Price of", product, ":", prices[choice])
        amount = int(input("Amount: "))
        if amount <= 0:
            print("Not a valid amount, please try again")
        order.append([choice, amount])
        print_order()
    print()
    print()
