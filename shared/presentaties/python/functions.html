What is a function

A function is a piece of code with a name.

To run the code, you call the function.

A function can (optionally):

  • Receive information (arguments)
  • Return information (return value)

Example: type

The builtin function type() takes a value and returns the type of that value: 

>>> x=5
>>> type(x)
type 'int'

This is better than having to copy/paste the source code for type all the time

Functions: why?

Functions prevent duplication. Instead of repeating the code, you give it a name and call it by that name.

Creating a function

def function_name( parameter1, parameter2, ... ):
    body

A function can take 0 or more parameters. The value passed to a parameter is an argument.

Example

def greet( firstname, lastname ):
    wholename = firstname + " " + lastname
    print("Hi, ", wholename )
  • Parameters firstname and lastname are local
  • Same for variables created inside the function (wholename)
  • Those variables cannot be reached outside the function.

But

What if you want to use the result of a function in the rest of your program? 

def multiply(x,y):
    print(x * y)

answer = multiply(5, 12)

The variable answer now has the value None!

Return

We can use the keyword return 

def multiply(x,y):
    return x * y

answer = multiply(5, 12)

Variable answer is 60 now and we can use that value for another calculation

Return 2

You might say that the return-value is substituted at the place of the function call. 

def half(x):
    return x / 2

print("half of 10 is", half(10))
l = [5, 8, half(12), 60]

Return

The return keyword does 2 things:

  • It ends the function; the program continues from where the function was called.
  • If you specify a return value, that is returned to the calling code.

No return

Functions without a return statement return the special value None.
