For: loop over elements of a list

Remember

A while loop repeats until something changes: 

x = 0
while x != -1:
    x = int(input("A number please? "))
    # do something with x

For

For repeats for every element in a sequence. 

for var in l:
    body
  • the body is executed once for every element of l
  • every time var takes the value of the next element of l

Example

Looping over every element of a list 

l = ["Jan", "Feb", "Mar", "Apr"]
for month in l:
    print(f"{month} starts with a {month[0]}")

Jan starts with a J
Feb starts with a F
Mar starts with a M
Apr starts with a A

Example 2

Looping over elements of a string 

count = 0
for character in "Today is a friday":
    if character == "a":
        count += 1
print(f"I counted {count} a's")

I counted 3 a's

For x in range

The function range() makes it possible to loop a fixed number of times: 

for i in range(100):
    # print all numbers from 0 to 99
    print(i)
