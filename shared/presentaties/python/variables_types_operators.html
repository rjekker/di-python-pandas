Types and Variables

Types

Value have a type. The most important types are:

  • int: 325, -8
  • float: 0.342
  • str: 'hello', or "hello"
  • bool: True or False

With strings you can use both single or double quotes. (what's the difference?)

Variables

Give a name to your value: 

>>> x = 3.5
>>> x
3.5
>>> x = "hello"

We call x a variable.

Variables are untyped: they can contain any kind of value.

Operators

Allow us to do calculations:

+, - add, subtract
*, /, // multiply, divide
** powers
% modulo

Use parens when it makes your code more clear: (3+5) / 2 = 4

Operators and Types

Update operators

Sometimes you want to update a variable value, for example: 

>>> x = 10
>>> x = x + 1
>>> x
11

# Equivalent
>>> x += 1
>>> x
12

Update operators 2

>>> s = "good"
>>> s += "morning"
>>> s
'goodmorning'

Update operators 3

This works for all basic operators: 

x += 1   # x = x + 1
y /= 5   # y = y / 5
z -= 10  # etc..
f *= 2
