Vowels

Write a program that asks the user to enter a letter and check whether it is a vowel or a consonant. If it is neither, you show an error message. 

# A little trick: we convert letters to lowercase
letter = input("Enter a letter?").lower()
if letter == 'a' or letter == 'e' or letter == 'i' or letter == 'o' or letter == 'u' or letter == 'y':
    print(letter, "is a vowel")
elif letter == 'b' or letter == 'c' or letter == 'd' or letter == 'f' or letter == 'g' or letter == 'h': # etc.
    print(letter, "is a consonant")
else:
    print(letter, "is neither a vowel or a consonant")

You can actually do that a bit more elegantly: 

letter = input("Enter a letter?").lower()
if letter in 'aeiouy':
    print(letter, "is a vowel")
elif letter in 'bcdfghjklmnpqrstvwxz': # etc.
    print(letter, "is a consonant")
else:
    print(letter, "is neither a vowel or a consonant")

  • 2.5, if 0 < w <= 2
  • 4.5, if 2 < w <= 4
  • 7.5, if 4 < w <= 10
  • 10.5, if 10 < w <= 20

Write a program that prompts the user to enter the weight of the package and display the shipping cost. If the weight is greater than 20, display a message â€œthe package cannot be shipped". 

w = int(input("Weight? "))

if 0 < w <=2:
    print("Cost: ", 2.5)
elif 2 < w <= 4:
    print("Cost: ", 4.5)
elif 4 < w <= 10:
    print("Cost: ", 7.5)
elif 10 < w <= 20:
