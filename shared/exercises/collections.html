Count words

Download the file words_with_a.txt. It contains a list of words starting with the letter a.

The question is: are there more words with length 5 than length 3? Which length is the most common? Which length is the least common?

Write a program that uses a Counter to determine this.

Use the following code to read the words from the file. 

with open("words_with_a.txt") as f:
    for line in f:
        word = line.strip()

Output: 

Counter({9: 3427, 10: 3209, 8: 3088, 11: 2756, ...})

Last n words

Adapt the same program to determine the last 5 words that end with the letter c. Use a deque for this.

Output: 

deque(['azosulphonic', 'azotemic', ...], maxlen=5)

Consider the following dataset:

NASDAQ = { 'MSFT': 100, 'AAPL': 200, 'TSLA': 80 }
AEX = {'ING': 50, 'RDSA': 70}
CRYPTO = {'BTC': 15000, 'ETH': 700}

Each stock exchange has its own stock symbols and prices. Write a program that enables the use to input a symbol (e.g. "MSFT") and search for that symbol in all stock exchanges. Use a ChainMap.

If the stock is not found, print an appropriate error.
