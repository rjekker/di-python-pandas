General Rules

Case Sensitive

Names (of variables, functions, etc.) are case sensitive.

So count and Count are two different variables!

Indentation has Meaning

Whitespace at the start of a line has meaning.

For example, it determines where blocks of code start and end.

Whitespace at the end of a line is ignored.

Comments

Any text starting with # is a comment and will be ignored.

Scripts

A Python Script is a text file, containing Python code, with a name ending in .py.

The Python interpreter will read each line from top to bottom and execute them.
