Some questions:

A tuple

Consider the following code: 

l = (1, 2, [3,4])

l is a tuple so it is immutable. What is the result of the following? 

l[2].append(5)

Try the code yourself to check.

A dict and a list

Consider the following: 

task_list = [ { 'worker': 1, 'tasks': [8,2,6,1] },
              { 'worker': 2, 'tasks': [5,7,3,9] }]

print([f"{w['worker']}: {w['tasks'].sort()}" for w in task_list])

What is the output you expect here? Run the code and test it.

What is the effect on the original task_list? How do we fix this?
