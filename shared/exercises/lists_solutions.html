Creating a list

Create a while loop for the following:

  • Ask the user for a word
  • Store this word in a list (use append)
  • Print the list
  • When the user enters nothing, stop the program
l = []
while True:
    word = input("A word please? ")
    if not word:
        break
    l.append(word)
    print(l)

Deduplication

Change the program so that, every time the user gives a word, you test whether it is already in the list. If it is, you don't add it. 

l = []
while True:
    word = input("A word please? ")
    if not word:
        break
    if word not in l:
        l.append(word)
    print(l)

Slices: step size

Slices can take a third argument: the step size. The following slice starts at the second element and skips 3 elements after that. 

l = list(range(20)) # numbers from 0 to 19
print(l[1::3])

Given the following list: 

l = list(range(3000))

Can you use slices to find the three largest multiples of 137 that are smaller than 3000? 

l = list(range(3000))
print(l[::137][-5:])

Output: 

[2329, 2466, 2603, 2740, 2877]

Complete the function parse_dollar_amount() so that it returns a tuple (dollars, cents) representing the dollars and cents in the input string amount.

Use the function str.index() to find the location of the dot (.) in the string. Then, use slices to determine dollar and cent amounts.

Example output: 

$2.45 is 2 dollars and .45 cents.
$10.80 is 10 dollars and .80 cents.
$100 is 100 dollars and 0 cents.
$18.3 is 18 dollars and .3 cents.
$.49 is 0 dollars and .49 cents.
$0.001 is 0 dollars and .001 cents.
$15.00 is 15 dollars and .00 cents.

def parse_dollar_amount(amount: str) -> tuple:
    """Parse the amount given as a string.
    Return a tuple (dollars, cents)."""
    try:
        index = amount.index(".")
        dollars = amount[1:index] or 0
        cents = amount[index:] or 0
        return (dollars, cents )
    except ValueError:
        dollars = amount[1:]
        return (amount[1:], 0)

if __name__=="__main__":
    for amount in ["$2.45", "$10.80", "$100", "$18.3", "$.49", "$0.001", "$15.00"]:
        dollars, cents = parse_dollar_amount(amount)
        print(f"{amount} is {dollars} dollars and {cents} cents.")
