Powers

Use a while loop to print the third power of all numbers between 0 and 100.

Then, change the code so that only the third power of every even number is printed.

Fizz Buzz

Implement Fizz Buzz:

The "Fizz-Buzz test" is an interview question designed to help filter out the 99.5% of programming job candidates who can't seem to program their way out of a wet paper bag. The text of the programming assignment is as follows:

"Write a program that prints the numbers from 1 to 100. But for multiples of three print “Fizz” instead of the number and for the multiples of five print “Buzz”. For numbers which are multiples of both three and five print “FizzBuzz”."

See http://www.c2.com/cgi/wiki?FizzBuzzTest

Class grades

Write a program that a teacher can use to compute the average grade for a test. Each time through the loop, the teacher enters a single grade.

When the teacher enters -1, you quit the loop, print the average and end the program.

Example: 

Enter a grade? 2
Enter a grade? 5
Enter a grade? 8
Enter a grade? -1
Average:  5.0

Hint:

  • you need a variable to hold the sum of all grades
  • you need another variable to count the number of grades
  • create both variables with a value of 0 at the start of the program
  • after the loop ends, do the calculation for the average grade

Best student

Can you add code to the previous program to show the highest score?

Example: 

Enter a grade? 1
Enter a grade? 8
Enter a grade? 9
Enter a grade? 2
Enter a grade? -1
Average:  5.0
Highest:  9

Simulation

Write a program that simulates tossing a coin a million times. Print the total number of heads and tails.

Example: 

500038 heads, and 499962 tails

Higher/Lower

Write the following game:

  • The computer chooses a random number (Use random.randint )
  • Ask the user for a number
  • Print a hint voor: "Higher!" or "Lower!"
  • Repeat this until the user guesses the number
