Random

Select a random link on the page and click it. Print the title and the current url of the page that you end up on.

Hint: see Browser Manipulation 

from selenium.webdriver import Chrome
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support.expected_conditions import url_changes

from random import sample

with Chrome() as driver:
    word = input("Word? ")

    driver.get("http://en.wikipedia.org/wiki/" + word)

    # First three paragraphs
    paragraphs = driver.find_elements_by_tag_name('p')
    for p in paragraphs[:3]:
        print(p.text)

    # Let's only search for links inside the actual content
    content = driver.find_element_by_xpath('//div[@id="mw-content-text"]')
    links = content.find_elements_by_tag_name('a')

    # select a random link
    l = sample(links, 1)[0]
    url = driver.current_url

    # click it, wait, print new url
    l.click()
    WebDriverWait(driver, 2).until(url_changes(url))

    print(driver.current_url)

Waiting

Ask the user for a stock name. When a price for that stock is shown on the page, print it and exit. 

from selenium.webdriver import Chrome
from selenium.webdriver.support.ui import WebDriverWait

def price_available(driver):
    price = driver.find_element_by_id(stock).text
    return price

with Chrome() as driver:
    driver.get("http://codesensei.nl/web/prices")

    # Let's follow apple prices
    stock = "aapl"

    WebDriverWait(driver, 2).until(price_available)

    print(driver.find_element_by_id(stock).text)

Ticker

Add a loop that waits for the price to change and prints the new prices as they appear. 

from selenium.webdriver import Chrome
from selenium.webdriver.support.ui import WebDriverWait

old_price = None

def price_changed(driver):
    price = driver.find_element_by_id(stock).text
    global old_price
    if price != old_price:
        old_price = price
        return price
    return False

with Chrome("/Users/reindert/bin/chromedriver") as driver:
    driver.get("http://codesensei.nl/web/prices")

    # Let's follow apple prices
    stock = "aapl"

    while True:
        WebDriverWait(driver, 10).until(price_changed)

        print(driver.find_element_by_id(stock).text)

Login

(This is a re-implementation of a previous exercise with requests) Write a script that goes to https://codesensei.nl/accounts/login/ and login with the following:

username nomura
password python

You will be rewarded with a nice message that appears on the page after login. Scrape this from the page and print it. 

from selenium.webdriver import Chrome
from selenium.webdriver.support.ui import WebDriverWait


with Chrome() as driver:
    driver.get("https://codesensei.nl/accounts/login")

    # Now let's find the inputs for the form
    user_input = driver.find_element_by_name('username')
    pwd_input = driver.find_element_by_name('password')
    # Best way to find submit button is by XPATH
    submit = driver.find_element_by_xpath('//input[@type="submit"]')

    user_input.send_keys("nomura")
    pwd_input.send_keys("python")
    submit.click()

    # There will be a page refresh after clicking submit
    # We wait for max. 5 seconds until the quote is on the page
    el = WebDriverWait(driver, 5).until(lambda d: d.find_element_by_class_name("quote"))

    print(driver.find_element_by_class_name("quote").text)
    print(driver.find_element_by_xpath('//p[@id="author"]').text)
