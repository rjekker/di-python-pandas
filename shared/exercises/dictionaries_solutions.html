Frequencies

Write a function frequencies(word) that counts the letters in a given word and returns frequency counts for every letter, as a dictionary: 

def frequencies(word):
    freq = {}
    for character in word:
        if character in freq:
            freq[character] += 1
        else:
            freq[character] = 1
    return freq

print(frequencies("good morning"))
print(frequencies("Antidisestablishmentarianism"))

Grades

Given the following dictionary with students and grades: 

{'John': [8, 2, 3, 6, 8],
 'Annie': [5, 8, 7, 8, 5],
 'Pete': [8, 8, 6, 7, 9],
 'Lucy': [2, 4, 5, 6, 7],
 'Bob': [6, 7, 5, 6, 7]}

Ask the user for the name of a student. Print the average grade for that student. 

class_scores = {'John': [8, 2, 3, 6, 8],
 'Annie': [5, 8, 7, 8, 5],
 'Pete': [8, 8, 6, 7, 9],
 'Lucy': [2, 4, 5, 6, 7],

student = input("Name of student? ")
if student in class_scores:
    # Retrieve the list of grades for this student
    grades = class_scores[student]
    print("Average for", student, ":", sum(grades)/len(grades))
else:

Class overview

Instead of asking for a single user, use a for loop to print the average grade for each student. 

for student in class_scores:
    # Retrieve the list of grades for this student
    grades = class_scores[student]
    print("Average for", student, "is:", sum(grades)/len(grades))

Adding grades

Ask the user for the name of a student and a grade. Add that grade to the list of grades for that student. Then print the new average.

Finally, print the entire dictionary. 

student = input("Add grade for student? ")
if student in class_scores:
    grade = int(input("New grade? "))
    # Retrieve the list of grades for this student
    grades = class_scores[student]
    # Append the new grade
    grades.append(grade)
    print("New average:", sum(grades)/len(grades))
    print(class_scores)
else:
    print("Unknown student")

Best Student

Print the name of the student with the highest average and the actual average for this student.

Hint: you need to loop over all the students. You also need two variables: one to keep track of the highest score you've seen, and one for the name of the corresponding student. 

best_score = -1
best_student = None
for student in class_scores:
    # Retrieve the list of grades for this student
    grades = class_scores[student]
    avg = sum(grades)/len(grades)
    if avg > best_score:
        best_student = student
        best_score = avg

print("Best student: ", best_student, "with score:", best_score)

Inventory management

Make it possible for the user to select a product and change the inventory.

The hard way: make the user type a product name, then find it in the dictionary. 

inventory = [ { 'name': 'bread', 'price': 2, 'stock': 100 },
  { 'name': 'coffee', 'price': 3, 'stock': 40 },
  { 'name': 'cheese', 'price': 3, 'stock': 30 },
  { 'name': 'milk', 'price': 1, 'stock': 80 } ]

def print_table():
    print("Name     Price    Stock")
    print("--------------------------")
    for prod in inventory:
        print(prod['name'], "\t", prod['price'], '\t', prod['stock'])

def find_prod_by_name(name):
    for prod in inventory:
        if prod['name'] == name.lower():
            return prod
    return None

print_table()
choice = input("Select a product? ")
product = find_prod_by_name(choice)
if product:
    amount = int(input("Amount in stock? "))
    product['stock'] = amount
    print_table()
else:
    print("Unknown product")

The easier way: let the user type the index of the product. 

inventory = [ { 'name': 'bread', 'price': 2, 'stock': 100 },
  { 'name': 'coffee', 'price': 3, 'stock': 40 },
  { 'name': 'cheese', 'price': 3, 'stock': 30 },
  { 'name': 'milk', 'price': 1, 'stock': 80 } ]

def print_table():
    print("id   Name       Price    Stock")
    print("-----------------------------")
    for id, prod in enumerate(inventory):
        print(str(id).ljust(4), prod['name'].ljust(10), prod['price'], '\t', prod['stock'])

print_table()
choice = int(input("Select a product? "))
if 0 <= choice < len(inventory):
    product = inventory[choice]
    amount = int(input("Amount in stock? "))
    product['stock'] = amount
    print_table()
else:
    print("Unknown product")
