Importing

When we import a package 

import my_pkg

What happens is determined by __init__.py. See here

Run a package

If the package has a __main__.py we can run it 

python -m my_package

This runs the code in __main__.py

Absolute vs relative

Absolute import

from my_pkg import something

Relative

Import relative to the "current" package. 

from . import something
from .. import something_else

(see also here)

In general

Absolute imports are preferred
