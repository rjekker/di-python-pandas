BankAccount

Given the BankAccount class: 

class BankAccount:
    def __init__(self, name, initial_balance):
        self.name = name
        self.balance = initial_balance

    def deposit(self, amount):
        if amount > 0:
            self.balance += amount
            return self.balance
        else:
            print("Can't deposit a negative amount!")

    def withdraw(self, amount):
        if amount > 0:
            if amount <= self.balance:
                self.balance -= amount
                return self.balance
            else:
                print("Insufficient balance!")
        else:
            print("Can't withdraw nonpositive amount!")
  • add __str__(), __repr__() methods
  • Add an =__eq__()__ method to the BankAccount class, such that two accounts are equal only if their balances AND owner names are equal. Make sure to test that your code works correctly!
  • Add a __mul__() method to the BankAccount class: this should create a new BankAccount with the names of both accounts added together, and the balances multiplied.
