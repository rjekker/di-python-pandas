Iterations (while)

Or: how to repeat yourself.

While

while boolean_expression:
    body

The code in the body will be repeated as long as the boolean_expression remains True.

There can be more than 1 line of code in the body.

1 to 100

count = 1
while count <= 100:
    print(count)
    count += 1

Break

break allows us to stop the loop whenever we want. 

count = 1
while True:
    print(count)
    if count >= 100:
        break
    count += 1

Break 2

while True:
    # allow user to do things
    # e.g. play a game, buy items, etc.
    answer = input("Want to continue [y/n]?")
    if answer == 'n':
        break
