Bonus: URL

This is an exercise for the more experienced students. For this exercise you need to learn about default arguments; see W3Schools or The official documentation.

Write a function generate_url(host):

If called with just a hostname, it will return a .com url. generate_url("pluralsight") returns http://www.pluralsight.com.

The function takes several optional arguments:

  • "tld": Set the top-level domain (.nl, .be, .com, etc.)
  • "secure": if True this will create a https url
  • "port": this will set the port number (see below)
  • "path": this will set the path of the url (part after the host/port

Example: 

print(generate_url('google', secure=True, port=4444, path="index.html", tld="fr"))

"https://www.google.fr:4444/index.html"

Please note that all arguments after the host name are optional!

Bonus: Craps

Craps is a popular dice game played in casinos. Write a program to play a variation of the game, as follows:

Roll two dice. Each die has six faces representing values 1, 2, . . ., and 6, respectively.

Check the sum of the two dice. If the sum is 2, 3, or 12 (called craps), you lose; if the sum is 7 or 11 (called natural), you win; if the sum is another value (i.e., 4, 5, 6, 8, 9, or 10), a point is established.

Continue to roll the dice until either a 7 or the same point value is rolled. If 7 is rolled, you lose. Otherwise, you win. Your program acts as a single player.
