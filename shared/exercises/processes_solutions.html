Arguments

Write a program calc.py that takes 3 arguments: two numbers and an operator. It should behave as follows: 

$ python calc.py 4 + 5
9

$ python calc.py 8 - 3
5

from sys import argv, exit

if len(argv) < 4:
    print("Need at least 3 arguments.")
    exit(1)

try:
    num1 = int(argv[1])
    num2 = int(argv[3])
except ValueError:
    print(f"One of the operands is not a number.")
    exit(2)

operator = argv[2]

if operator == "+":
    print(num1 + num2)
elif operator == "-":
    print(num1 - num2)
else:
    print(f"Unknown operator {operator}")
    exit(3)

Errors

Catch exceptions and print appropriate errors when:

  • arguments are missing
  • too many arguments are passed
  • arguments are not of the correct type

If one of those errors occurs, exit with status code 1.

If everything is ok, exit with status code 0. 

from sys import argv, exit

def check_arg_length():
    """Print appropriate messages for number of args"""
    if len(argv) == 4:
        return
    if len(argv) < 4:
        print("Not enough arguments")
    elif len(argv) > 4:
        print("Too many arguments")
    exit(1)


def get_int_arg(n):
    """Get nth argument, which should be a number, convert it to int and return it."""
    try:
        return int(argv[n])
    except:
        print(f"argument {n}: '{argv[n]}' is not a number")
        exit(1)

def do_calculation():
    a = get_int_arg(1)
    op = argv[2]
    b = get_int_arg(3)
    if op == "-":
        return a-b
    elif op == "+":
        return a+b
    else:
        print(f"Unknown operator '{op}'")
        exit(1)

if __name__ == "__main__":
    check_arg_length()
    print(do_calculation())

IP address

Write a python program that uses subprocess.run() to call the command ipconfig (on Mac/Linux: ifconfig).

Process the output and print only your IP address. 

import locale
import subprocess

c = subprocess.run(['ifconfig'], stdout=subprocess.PIPE, text=True)
lines = c.stdout.split("\n")
for l in lines:
    words = l.split(' ')
    # Mac OS is simple
    if words[0].strip() == "inet":
        print(words[1])

    # Windows is harder
    try:
        if words[3].strip().startswith("IPv4"):
            print(words[-1])
    except:
        continue

Ping Sweep

Download the file ../static/hosts.txt.

Write a python scripts that:

  • reads the file
  • pings each host in the file
  • checks the result of the ping command
  • prints a nice overview
import subprocess

with open("../../static/hosts.txt") as hosts:
    for host in hosts:
        host = host.strip()
        c = subprocess.run(['ping', '-c', '3', host], capture_output=True)
        if not c.returncode:
            print(f"{host} is UP")
        else:
            print(f"{host} appears to be DOWN")
