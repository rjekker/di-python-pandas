Interfaces: Reading

Download interfaces.csv and write a python program to do the following:

  • Open the file
  • Create a csv.reader
  • for each server in the list: print the command to configure the ip and subnet for that server.
import csv

with open("../../static/interfaces.csv") as f:
    f.readline() # skip header
    r = csv.reader(f)
    for server in r:
        print(f"interface {server[1]}")
        print(f"ip address {server[2]} {server[4]}")
        print("-----\n\n")

Interfaces: Filtering

Change your program so that it asks the user for a vlan number. Print only the data for the lines that match that vlan.

Add error handling for incorrect user input.

Can you add logic to print an error when the user input is a number, but there are no matching lines? 

import csv

try:
    vlan = int(input("Vlan number? "))

    matching = False

    with open("../../static/interfaces.csv") as f:
        f.readline() # skip header
        r = csv.reader(f)
        for server in r:
            if int(server[1]) == vlan:
                matching = True
                print(f"interface {server[1]}")
                print(f"ip address {server[2]} {server[4]}")
                print("-----\n\n")

    if not matching:
        print("No matching lines")
except ValueError:
    print("That is not a valid number.")

Animal classes

All the animals in the zoo have a class (this is the last column in the csv). Here's a dictionary with the class numbers: 

classes = {
  "Mammal": 1,
  "Bird": 2,
  "Reptile": 3,
  "Fish": 4,
  "Amphibian": 5,
  "Bug": 6,
  "Invertebrate": 7
}

Write a program that:

  • Asks the user for a class (e.g. "Fish")
  • Reads zoo.csv
  • Print the names for the animals in that class
  • Make sure to print a nice message when the user input is not an existing class!

Note: you cannot use the word class in Python as a variable, it is a reserved word! 

import csv
clazz = input("Animal class? ")

classes = {
    "Mammal": 1,
    "Bird": 2,
    "Reptile": 3,
    "Fish": 4,
    "Amphibian": 5,
    "Bug": 6,
    "Invertebrate": 7
}

try:
    class_nr = classes[clazz]
except KeyError:
    print("Unknown class!")

with open("../../static/zoo.csv") as f:
    r = csv.DictReader(f)
    for animal in r:
        if int(animal['class_type']) == class_nr:
            print(animal['animal_name'])

Writing CSV files

Instead of printing the names of the animals in a class, change your program so that:

  • It creates a list with all the names for the animals in a class
  • Writes that data to a new file (e.g. "Fish.csv").
import csv
clazz = input("Animal class? ")

classes = {
    "Mammal": 1,
    "Bird": 2,
    "Reptile": 3,
    "Fish": 4,
    "Amphibian": 5,
    "Bug": 6,
    "Invertebrate": 7
}

try:
    class_nr = classes[clazz]
except KeyError:
    print("Unknown class!")

animals = []

with open("../../static/zoo.csv") as f:
    r = csv.DictReader(f)
    for animal in r:
        if int(animal['class_type']) == class_nr:
            animals.append(animal)

with open(clazz + ".csv", "w") as f2:
    w = csv.DictWriter(f2, r.fieldnames)
    w.writeheader()
    for animal in animals:
        w.writerow(animal)

Bonus: all classes

Instead of asking the user for a class, create a loop that writes a file for every animal class. So this would create 7 files: Fish.csv, Mammal.csv, etc. 

import csv

classes = {
    "Mammal": 1,
    "Bird": 2,
    "Reptile": 3,
    "Fish": 4,
    "Amphibian": 5,
    "Bug": 6,
    "Invertebrate": 7
}

animals = []

with open("../../static/zoo.csv") as f:
    r = csv.DictReader(f)
    for animal in r:
        animals.append(animal)

for clazz in classes.keys():
     with open(clazz + ".csv", "w") as f2:
         w = csv.DictWriter(f2, r.fieldnames)
         w.writeheader()
         for animal in animals:
             if int(animal['class_type']) == classes[clazz]:
                 w.writerow(animal)
