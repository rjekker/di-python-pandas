Text files and I/O

Open

Built-in function open(filename, mode)

Returns a File object. 

fin = open("somefile.txt")
for l in fin:
    print(l)

Mode

r read from start of file (default)
r+ read and write from start
w truncate; write from start
w+ read and write; truncate
a write at end of file
a+ read and write at end of file
  • It's usually better not to use the modes with a +

File Output

fout = open("file.txt", 'a')
print("hello, world!", file=fout)

#alternative
fout.write("hello, world!")

print() vs f.write()

  • print() adds a newline, f.write() does not
  • print() calls __str__() on an object

closing files

Best practice: to make sure file is closed on errors 

with open("log.txt") as f:
    for l in f:
        print(l)
  • This is like wrapping the code with try/except and calling f.close
