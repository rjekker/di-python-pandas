Warming up

Create a dictionary with keys "name", "age" and "city". For each key, fill in the values for yourself (e.g. "Reindert", 42, "Amsterdam").

Write some code that adds a key "phone" with your phone number.

Then, add a line that changes the city into another city you would like to live in.

print the results.

Example output: 

{'name': 'Reindert', 'age': 42, 'city': 'Groningen', 'phone': '+31612345678'}

Ip adresses

Write a program to store a list of servers with their ip-adresses. Store this in a dictionary and print the dictionary every time the user enters a new piece of data. The result should look like this: 

Server name? dns
Ip address? 8.8.8.8
{'dns': '8.8.8.8'}
Continue (y/n)y
Server name? home
Ip address? 127.0.0.1
{'dns': '8.8.8.8', 'home': '127.0.0.1'}
Continue (y/n)y
Server name? router
Ip address? 192.168.1.254
{'dns': '8.8.8.8', 'home': '127.0.0.1', 'router': '192.168.1.254'}
Continue (y/n)n

Adressbook

Part 1: the data

You want to write a program to store and manage the addresses of your friends and relatives.

Create a dictionary that takes the name of a friend or relative as the key, and their address as value. Add the addresses of at least 3 people.

Part 2: Retrieve addresses

Add code to ask the user for the name of a person and then print the corresponding address.

If the person is not in the address book, print a nice and informative message.

Example: 

Contact name? Pluralsight
Pluralsight :
------------
182 N. Union Ave, Farmington, UT 84025


Contact name? RJ
Unknown contact

Grades

Given the following dictionary with students and grades: 

{'John': [8, 2, 3, 6, 8],
 'Annie': [5, 8, 7, 8, 5],
 'Pete': [8, 8, 6, 7, 9],
 'Lucy': [2, 4, 5, 6, 7],
 'Bob': [6, 7, 5, 6, 7]}

Ask the user for the name of a student. Print the average grade for that student. If the name given is not in the dictionary, print a friendly error message. 

Name of student? Bob
Average for Bob : 6.2

Name of student? RJ
Unknown student :(

Class overview

Instead of asking for a single user, use a for loop to print the average grade for each student. 

Average for John is: 5.4
Average for Annie is: 6.6
Average for Pete is: 7.6
Average for Lucy is: 4.8
Average for Bob is: 6.2

Adding grades

Ask the user for the name of a student and a grade. Add that grade to the list of grades for that student. Then print the new average.

Finally, print the entire dictionary. 

Add grade for student? Bob
New grade? 8
New average: 6.5
{'John': [8, 2, 3, 6, 8], 'Annie': [5, 8, 7, 8, 5], 'Pete': [8, 8, 6, 7, 9], 'Lucy': [2, 4,
 5, 6, 7], 'Bob': [6, 7, 5, 6, 7, 8]}

Best Student

Print the name of the student with the highest average.

Print the name of the student with the highest average and the actual average for this student.

Hint: you need to loop over all the students. You also need two variables: one to keep track of the highest score you've seen, and one for the name of the corresponding student.

Frequencies

Write a function frequencies(word) that counts the letters in a given word and returns the counts for every letter, as a dictionary.

To test, call the function with a couple of words, like this: 

print(frequencies("good morning"))
print(frequencies("Antidisestablishmentarianism")

output: 

{'g': 2, 'o': 3, 'd': 1, ' ': 1, 'm': 1, 'r': 1, 'n': 2, 'i': 1}
{'A': 1, 'n': 3, 't': 3, 'i': 5, 'd': 1, 's': 4, 'e': 2, 'a': 3, 'b': 1, 'l': 1, 'h': 1, 'm
': 2, 'r': 1}

Hint: you can loop over the characters of a string just like you would loop over the elements in a list, like this. 

s = "hello there!"
for character in s:
    print(character)

Another store

Given the following store inventory: 

[ { 'name': 'bread', 'price': 2, 'stock': 100 },
  { 'name': 'coffee', 'price': 3, 'stock': 40 },
  { 'name': 'cheese', 'price': 3, 'stock': 30 },
  { 'name': 'milk', 'price': 1, 'stock': 80 } ]

Print the inventory as a table: 

Name     Price    Stock
--------------------------
bread    2       100
coffee   3       40
cheese   3       30
milk     1       80

Hint: print a tab character \t between each column to make them line up nicely. Another solution would be to use f-strings.
