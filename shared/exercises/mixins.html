JavaScript for loop

JavaScript allows looping over any object; this will loop over all the properties of that object.

Implement a Mixin JSIterable that implements this. 

class JSIterable:
    # Your code here..


class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

class IterablePerson(JSIterable, Person):
    pass

p = IterablePerson("RJ", 43)
for prop in p:
    print(prop, p.__getattribute__(prop))

output: 

name RJ
age 43

Role playing

Let's create a role playing game. Characters classes are derived from a base class Character which you can see below.

We create new character classes with mixins:

  • a mixin CloseCombat which implements attack by hitting someone with a club. Damage is equal to strength
  • a mixin Magic which implements attacking by casting a spell. Damage is equal to magic
  • a mixin Strong which doubles a character's strength

A Barbarian has both CloseCombat and Strong. A wizard has Magic.

Log all calls

Create a mixin class LogAllCalls that makes sure that every call to every method on the object is logged.

Hint: to test if something is callable, use hasattr to test for a method __call__. Next you have to figure out how to make sure to log all calls to that method.

Example use: 

# Using the mixin
class MyClass(LogAllCalls):
    def __init__(self):
        self.i = 10

    def x(self, x, y):
        return x+y

    def y(self, a, b, c):
        return a*b*c

obj = MyClass()
print(obj.i) # no logging

# following are function calls: should be logged
print(obj.x(10,20))
print(obj.y(c=1,b=2,a=3))

class MyStr(LogAllCalls,str):
    pass

x = MyStr("hello")
print(x.upper())

output: 

10
Calling x with args: (10, 20), {}
30
Calling y with args: (), {'c': 1, 'b': 2, 'a': 3}
6
Calling upper with args: (), {}
