Adding data

Add code to your program that asks for the city the person lives in. Can you add this to the data for that person?

You can do this, but you have to change the datastructure into a list! 

persons = [["Reindert", "Ekker", 25],
           ["Joris", "Hoendervangers", 24],
           ["Jeroen", "Meijer", 23]]

def get_person(index):
    # Please add code here
    person = persons[index]
    return (person[0], person[1], person[2])

num = int(input("Give a number [0-2]? "))
firstname, lastname, age = get_person(num)
print(f"This person is called {firstname} {lastname} and is {age} years old.")
city = input("Where does this person live?")
persons[num].append(city)
print(persons)

Store

Given the following datastructure with store inventory: 

[ { 'name': 'bread', 'price': 2, 'stock': 100 },
  { 'name': 'coffee', 'price': 3, 'stock': 40 },
  { 'name': 'cheese', 'price': 3, 'stock': 30 },
  { 'name': 'milk', 'price': 1, 'stock': 80 } ]

Write a program that prints the name and price of each product, and the index in the list. Use the enumerate() method for this. (see tutorial).

Example output: 

0: bread
1: coffee
2: cheese
3: milk

products = [ { 'name': 'bread', 'price': 2, 'stock': 100 },
    { 'name': 'coffee', 'price': 3, 'stock': 40 },
    { 'name': 'cheese', 'price': 3, 'stock': 30 },
    { 'name': 'milk', 'price': 1, 'stock': 80 } ]

for index, prod in enumerate(products):
    print(f"{index}: {prod['name']}")

Items

Add code to ask the user to select a product by entering a number.

Next, use dict.items() to loop over the dictionary and print the data for that product.

Add error handling for when the user enters incorrect data.

Example output: 

0: bread
1: coffee
2: cheese
3: milk
Your choice? 3
name: milk
price: 1
stock: 80

products = [ { 'name': 'bread', 'price': 2, 'stock': 100 },
    { 'name': 'coffee', 'price': 3, 'stock': 40 },
    { 'name': 'cheese', 'price': 3, 'stock': 30 },
    { 'name': 'milk', 'price': 1, 'stock': 80 } ]

for index, prod in enumerate(products):
    print(f"{index}: {prod['name']}")

try:    
    num = int(input("Your choice? "))
    prod = products[num]
    for key, val in prod.items():
        print(f"{key}: {val}")
except IndexError:
    print("No such product")
except ValueError:
    print("That is not a number")
