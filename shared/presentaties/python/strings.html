Printing

Remember

We can construct a string to print using the + operator: 

print("5 + 5 = " + str(10))

Comma's are better

Using a comma, print will send each value to the output without having to convert them. 

print("5 + 5 =", 10, "!")

Note that print will add a space between the values.

f-strings

This is probably the best way to construct strings.

Put an f in front of the string and use your variables inside it. 

name = "John"
grade = 8
print(f"Student {name} has grade {grade}")

Student John has grade 8

Expressions in f-strings

You can put simple expressions in f-strings 

price = 5
items = 4
print(f"the total price for {items} items is {price*items} euros.")

the total price for 4 items is 20 euros.

Using lists

You can even use lists or dictionaries in an f-string: 

week = ["Monday", "Tuesday", "Wednesday", ...]
print(f"The second day is {week[2]}")

The second day is Wednesday

Sequence

A string is a sequence, like a list. This means it also has a length and you can use indices to get its elements.

Length

print(len("hello there"))

11

Indices

name = "Reindert"
print(name[0])
print(name[-1])

R
t

Slices

date = "2021-02-13"
year = date[:4]
month = date[5:-3]
day = date[-2:]
print(year, month, day)

2021 02 13

But

You cannot change elements of a string! 

s = "apple"
s[0] = "A"

TypeError: 'str' object does not support item assignment

Similarly

We cannot append to a string. 

s = "apple"
s.append("s")

AttributeError: 'str' object has no attribute 'append'

String operations

Strings are objects, just like lists. They have their own functions, like upper() and lower(). 

next_game = input("continue (y/n)? ")
if next_game.upper() == 'N':
    exit()

Note

There's no need to know all of these by heart. Google is your friend!

Splitting

split() returns a list of strings. 

s = "man,woman,camera,tv"
l = s.split(",")
print(l)
print(l[2])

['man', 'woman', 'camera', 'tv']
camera
