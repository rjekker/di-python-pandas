Length

Inspect the page at https://codesensei.nl/web/form-demo/.

Write a program that asks the user for a string, sends this to the server, and prints the length that the server computed. 

import requests
from bs4 import BeautifulSoup

text = input("Please enter some text: ")

# Create a session; set necessary headers
session = requests.Session()
session.headers = {"Referer":"https://codesensei.nl"}

# First we get the page with the form
resp = session.get("https://codesensei.nl/web/form-demo/")
soup = BeautifulSoup(resp.text, 'html.parser')

# Get the CSRF token
csrf = soup.find('input', {'name':"csrfmiddlewaretoken"})['value']

# Post our text
data = data={'text': text, 'csrfmiddlewaretoken': csrf }
resp = session.post("https://codesensei.nl/web/form-demo/",
                    data = data
                    )

# Parse the response
soup = BeautifulSoup(resp.text, 'html.parser')
print('length:', soup.find('span', {'id': 'length'}).text)

Chat

Inspect the page at https://codesensei.nl/web/chat/.

We can chat with each other by filling in the form. Send some messages to try this.

We want to create a commandline chat client for this chat.

Listing messages

Start by scraping the page and printing the messages so far. 

import requests
from bs4 import BeautifulSoup

# Create a session; set necessary headers
session = requests.Session()
session.headers = {"Referer":"https://codesensei.nl"}

# Get the page
resp = session.get("https://codesensei.nl/web/chat/")
soup = BeautifulSoup(resp.text, 'html.parser')

# Print all existing messages
for message in soup.find_all('p', class_="message"):
    print(message.find('strong').text, ":", message.find('span').text)

Posting messages

Then, ask the user for their name and message, and send a POST request to post the message.

Check that the message is actually shown on the page.

(Please note that there are no restrictions in place so please don't spam this page with thousands of messages.) 

# First part is identical to the script above
import requests
from bs4 import BeautifulSoup

# Create a session; set necessary headers
session = requests.Session()
session.headers = {"Referer":"https://codesensei.nl"}

# Get the page
resp = session.get("https://codesensei.nl/web/chat/")
soup = BeautifulSoup(resp.text, 'html.parser')

# Print all existing messages
for message in soup.find_all('p', class_="message"):
    print(message.find('strong').text, ":", message.find('span').text)

# --- now let's post a message ---
# Get the CSRF token
csrf = soup.find('input', {'name':"csrfmiddlewaretoken"})['value']

user = input("Your name? ")
msg = input("Your message? ")

resp = session.post("https://codesensei.nl/web/chat/",
                  data=dict(username=user,
                            text=msg,
                            csrfmiddlewaretoken=csrf),
                  )

Login

Write a script that goes to https://codesensei.nl/accounts/login/ and login with the following:

username nomura
password python

Print the message that appears on the page after login. 

import requests
from bs4 import BeautifulSoup

# Create a session; set necessary headers
session = requests.Session()
session.headers = {"Referer":"https://codesensei.nl"}

# Get the page
resp = session.get("https://codesensei.nl/accounts/login/")
soup = BeautifulSoup(resp.text, 'html.parser')

# Get the CSRF token
csrf = soup.find('input', {'name':"csrfmiddlewaretoken"})['value']

resp = session.post("https://codesensei.nl/accounts/login/",
                  data=dict(username="nomura",
                            password="python",
                            csrfmiddlewaretoken=csrf),
                  )
soup = BeautifulSoup(resp.text, 'html.parser')
print(soup.find('p', class_='quote').text)
print(soup.find('p', {'id': 'author'}).text)
