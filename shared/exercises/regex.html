Simple match

Check whether the given strings contain 0xB0. Print True or False.

Example input: 

line1 = 'start address: 0xA0, func1 address: 0xC0'
line2 = 'end address: 0xFF, func2 address: 0xB0'

Replace

Replace all occurrences of 5 with five for the given string.

Example input: 

ip = 'They ate 5 apples and 5 oranges'

Output: "They ate five apples and five oranges"

Only once

Change your code to only replace the first occurence. Output: "They ate five apples and 5 oranges"

Filter list

For the given list, return all elements that do not contain e. 

items = ['goal', 'new', 'user', 'sit', 'eat', 'dinner']

Case insensitive

For the given input string, display all lines not containing start irrespective of case. 

para = """good start
    Start working on that
    project you always wanted
    stars are shining brightly
    hi there
    start and try to
    finish the book
    bye"""

Another list filter

For the given list, return only elements that contains either a or w. 

items = ['goal', 'new', 'user', 'sit', 'eat', 'dinner']

Both

Match all elements that contains both e and n. 

items = ['goal', 'new', 'user', 'sit', 'eat', 'dinner']

Start anchor

Check if the given strings start with be. 

line1 = 'be nice'
line2 = '"best!"'
line3 = 'better?'
line4 = 'oh no\nbear spotted'

Whole word

For the given input string, change only the whole word red to brown 

words = 'bred red spread credible'

Expected output: 'bred brown spread credible'

Whole word at start

For the given input string, change whole word mall to 1234 only if it is at the start of a line. 

para = """\
    ball fall wall tall
    mall call ball pall
    wall mall ball fall
    mallet wallet malls"""

Re, re

For the given input strings, replace all occurrences of removed or reed or received or refused with X. 

s1 = 'creed refuse removed read'
s2 = 'refused reed redo received'

Quantifiers

For the list items, filter all elements starting with hand and ending with at most one more character or le. 

items = ['handed', 'hand', 'handled', 'handy', 'unhand', 'hands', 'handle']

Quantifiers 2

For the given input strings, remove everything from the first occurrence of i till end of the string. 

s1 = 'remove the special meaning of such constructs'
s2 = 'characters while constructing'

Test

For the given input strings, remove everything from the first occurrence of test (irrespective of case) till end of the string, provided test isn't at the end of the string. 

s1 = 'this is a Test'
s2 = 'always test your RE for corner cases'
s3 = 'a TEST of skill tests?'

Test2

For the input list words, filter all elements starting with s and containing e and t in any order. 

words = ['sequoia', 'subtle', 'exhibit', 'asset', 'sets', 'tests', 'site']

Six

For the input list words, remove all elements having less than 6 characters. 

words = ['sequoia', 'subtle', 'exhibit', 'asset', 'sets', 'tests', 'site']

Group

For the given strings, extract the matching portion from first is to last t. 

str1 = 'This the biggest fruit you have seen?'
str2 = 'Your mission is to read and practice consistently'

Output: 

'is the biggest fruit'
'ission is to read and practice consistent'

Parens

Extract all words between ( and ) from the given input string as a list. Assume that the input will not contain any broken parentheses. 

ip = 'another (way) to reuse (portion) matched (by) capture groups'x

Tags

Extract all occurrences of < up to next occurrence of >, provided there is at least one character in between < and >. 

ip = 'a<apple> 1<> b<bye> 2<> c<cat>'

Character class

Filter all elements whose first non-whitespace character is not a # character. Any element made up of only whitespace characters should be ignored as well. 

items = ['    #comment', '\t\napple #42', '#oops', 'sure', 'no#1', '\t\r\f']

Backref

Add [] around words starting with s and containing e and t in any order. 

ip = 'sequoia subtle exhibit asset sets tests site'

##### add your solution here
'sequoia [subtle] exhibit asset [sets] tests [site]'
