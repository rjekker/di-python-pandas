Installeren

Gebruik pip install netmiko of installeer netmiko in je pycharm project.

Maken van een verbinding

Importeer netmiko, maak een ConnectHandler aan en geef mee:

  • host
  • username
  • device type (zie supported devices hieronder)
  • password
  • zonodig secret (voor enable())
from netmiko import ConnectHandler

username = input()
password = input()
secret = input()

net_connect = ConnectHandler(device_type='cisco_xe',
                             host='example.com',
                             username=username,
                             password=password,
                             secret=secret)

Veilig wachtwoorden invoeren

Hiervoor kun je getpass gebruiken: 

import getpass
wachtwoord = getpass.getpass("geef wachtwoord? ")

Let op: dit werkt niet in pycharm, wel als je je script vanaf de command line uitvoert.

Niet vergeten aan het eind van je script

net_connect.disconnect()

Uitvoeren commando's

Gebruik send_command om een commando uit te voeren. Het resultaat is een lange string die we moeten opsplitsen in substrings. 

output = net_connect.send_command("show arp")
lines = output.split('\n')
for l in lines:
    if "CONNECTED" in l: 
        print(l)

Enable

Voor sommige commando's moet je eerst een enable uitvoeren: 

net_connect.enable()

In dit geval is het wel belangrijk dat je bij het aanmaken van de connectie een secret hebt meegegeven.

Configuratie-commando's

Commando's die in config mode ("conf t") moeten worden uitgevoerd, kunnen we uitvoeren met send_config_set. Deze functie accepteert een lijst met commando's of een string met meerdere regels.

Voorbeeld met meerdere regels

net_connect.send_config_set("int GigabitEthernet1/0/1\ndesc test_omschrijving")

Hetzelfde voorbeeld met een lijst:

net_connect.send_config_set(["int GigabitEthernet1/0/1",
                            "desc test_omschrijving"])

Check

Na het uitvoeren van de bovenstaande interface configuratie, kunnen we de configuratie voor de interface op de volgende manier printen: 

output = net_connect.send_command("show run")
lines = output.split('\n')

# loop over alle regels in de output
# maar voor iedere regel halen we ook meteen de index op
for index, line in enumerate(lines):
    if "GigabitEthernet1/0/1" in line:
        # Interface gevonden
        # Nu printen we alles tot aan de "!"
        while lines[index] != "!":
            print(lines[index])
            index += 1

Config file uploaden

Hiervoor gebruik je send_config_from_file: 

net_connect.send_config_from_file(os.path.join("configuraties", "config2.txt"))

Netmiko: supported devices

  • cisco_ios, cisco_xe, cisco_asa, cisco_nxos, cisco_xr, cisco_wlc_ssh
  • arista_eos
  • hp_procurve, hp_comware
  • huawei
  • f5_ltm
  • juniper
  • brocade_vdx
