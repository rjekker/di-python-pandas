Codesensei Pages and Status Codes

Use request to retrieve the following urls:

  • https://codesensei.nl
  • https://codesensei.nl/courses/intro
  • https://codesensei.nl/courses/nomura-advanced

Which different status codes do each of these pages return and why? 

for url in ["https://codesensei.nl", "https://codesensei.nl/courses/intro", "https://codesensei.nl/courses/nomura-advanced"]:
    response = requests.get(url) # Don't forget to add proxy settings if needed
    print(response.status_code)

Google search

Write a program that asks the user for a word and then does a google search for that word.

To do this, you send a request to https://www.google.com/search. This page accepts a request parameter q, which is the term you search for.

Check the contents of the response. Soon we will see how to extract useful information from this page. 

word = input("What shall I search for? ")
response = requests.get("https://www.google.com/search", params={'q': word})
print(response.text)

Basic Authentication

Try to retrieve the page at https://codesensei.nl/courses/auth-demo/

Which status code do you get? Answer: 401

Can you retrieve the page by authenticating as nomura with password "python"?

Answer: yes, using the following: 

requests.get(
    "https://codesensei.nl/courses/auth-demo/", auth=("nomura", "python"))
