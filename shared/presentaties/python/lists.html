Lists

Lists

Lists contain 0 or more elements:

  • Of any type
  • You can add, remove and change the elements
>>> [1, 2, "hello", [5, 6]]
[1, 2, 'hello', [5, 6]]

Creating a list

  • [2.5, "a", 1]
  • The empty list: []

len

The builtin function len() retrieves the length of the list: 

len([8, 'a', [], '35'])

Indices

The nth element can be retrieved with square brackets: l[5] 

l = ['tuesday', 'wednesday', 'friday']
print(l[1])

wednesday
  • The index of the first element is 0!

Negative indices

Negative index counts from the end backwards: 

l = ["Ann", "Bob", "Chloe", "Dan", "Ella"]
print(l[-1])
print(l[-3])

Ella
Chloe

Change an element

You can assign a new value or update an existing item 

numbers = [0, 1, 2, 3, "four", 4]
numbers[4] = 4
numbers[5] += 1
print(numbers)

[0, 1, 2, 3, 4, 5]

Remove an element

You can remove an element using del, or remove 

l = ["h", "!", "b", "i", "!"]
del l[2] # remove element at index 2
l.remove("!") # remove first element with value "!"
print(l)

['h', 'i', '!']

Insert at position

l = ["a", "b", "d"]
l.insert(2, "c")
print(l)

['a', 'b', 'c', 'd']

Slices

Slicing means to take a subsequence from a sequence. The syntax is s[start:end]. Where start and end are the indexes of the subsequence you want.

Example 1

Note: Slices do not include the end value. 

l = ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug"]
print(l[2:4])
print(l[-4:-1])

['Mar', 'Apr']
['May', 'Jun', 'Jul']

Example 2

If indices are omitted, they default to the beginning or end of the list. 

l = ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug"]
print(l[:3])
print(l[-3:])

['Jan', 'Feb', 'Mar']
['Jun', 'Jul', 'Aug']

Note

Slices and indices work on strings as well 

amount = "$4.53"
print(amount[0])
print(amount[-2:])

$
53

Finding elements

In

Check if something is in the list with in and not in: 

l = [2, 4, 6, 8, 10]
if 6 in l:
    print("6 is in l")
if 9 not in l:
    print("9 is not in l")

6 is in l
9 is not in l

Index

Find the position of an element 

l = ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug"]
print(l.index("Mar"))

2

Count

Count how often an element appears in the list 

l = [1, 1, 2, 1, 2, 2, 2, 1, 1, 1, 2, 2, 2]
print(l.count(2))
