Flask

https://flask.palletsprojects.com

A microframework for building websites in Python

Minimal Application

from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

Running

Commandline: 

flask --app mysite run --debug
  • Run this from project dir
  • mysite is the name of the python file containing the app object
  • --debug makes sure that the app reloads on changes

Templates

HTML Pages

See for instance https://www.w3schools.com/html/html_intro.asp 

from flask import Flask, render_template
app = Flask(__name__)

@app.route('/')
def hello_world():
    return render_template("hello.html")

The html file hello.html must be in a folder templates

Pass data to template

@app.route('/')
def hello_world():
    return render_template("hello.html",
        name="reindert", age=43)

Template syntax

Double curly braces: output a variable value 

<p> Hi, {{ name }}!</p>

Statements like if and for go inside {% .. %} 

<ul>
{% for user in users %}
    <li>{{ user.username|e }}</li>
{% endfor %}
</ul>
