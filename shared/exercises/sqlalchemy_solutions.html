Car Sharing

Let's create an application for a car-sharing platform. We have two classes, each with its own table:

  • Car: this has at least a brand, a model, a license id, and the year the car was built.
  • Journey: a single trip taken by a car. Contains a distance and a description.
  • A Journey has a Car (foreignkey)

Create persistent classes for both of these. Also make sure that there are relations (see relations and the example code here)

Write some code that creates one or more cars. Allow the user to enter a journey by selecting a car and entering a distance and a description.

Finally, make sure that Car has a property total_kms which returns the total number of kilometers for all journeys. 

from sqlalchemy import create_engine
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy import Column, Integer, String, ForeignKey
from sqlalchemy.orm import sessionmaker, relationship

engine = create_engine('sqlite:///cars.db', echo=True)

Base = declarative_base()


class Car(Base):
    __tablename__ = "cars"  # DB tabelnaam

    id = Column(Integer, primary_key=True)
    brand = Column(String)
    model = Column(String)
    license = Column(String)
    year = Column(Integer)

    journeys = relationship("Journey", back_populates="car")

    @property
    def total_kms(self):
        return sum(j.distance for j in self.journeys)

    def __str__(self):
        return f"{self.license}: {self.total_kms}"


class Journey(Base):
    __tablename__ = "journeys"  # DB tabelnaam

    id = Column(Integer, primary_key=True)
    distance = Column(Integer)
    description = Column(String)
    car_id = Column(Integer, ForeignKey('cars.id'))
    car = relationship("Car", back_populates="journeys")

    def __str__(self):
        return f"{self.description}: {self.distance}"


if __name__ == "__main__":
    Base.metadata.create_all(engine)
    # 1 sessionfactory per python programma/db connection
    session_factory = sessionmaker(bind=engine)
    # Create a session for each transaction
    session = session_factory()
    car = session.query(Car).filter(Car.model=="astra").one()
    print("---")

    print(car)
