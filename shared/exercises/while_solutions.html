Powers

Use a while loop to print the third power of all numbers between 0 and 100. 

i = 0
while i < 101:
    print(i*i*i)
    i += 1

Then, change the code so that only the third power of every even number is printed. 

i = 0
while i < 101:
    print(i, i*i*i)
    i += 2

Or another solution: 

i = 0
while i < 101:
    if i % 2 == 0:
        print(i*i*i)
    i += 2

Fizz Buzz

"Write a program that prints the numbers from 1 to 100. But for multiples of three print “Fizz” instead of the number and for the multiples of five print “Buzz”. For numbers which are multiples of both three and five print “FizzBuzz”." 

count = 1
while count <= 100:
    if count % 3 == 0:
        if count % 5 == 0:
            print("FizzBuzz")
        else:
            print("Fizz")
    elif count % 5 == 0:
        print("Buzz")
    else:
        print(count)
    count += 1

Or, a slightly more elegant solution: 

count = 1
while count <= 100:
    line = ""
    if count % 3 == 0:
        line += "Fizz"
    if count % 5 == 0:
        line += "Buzz"
    if line:
        print(line)
    else:
        print(count)
    count += 1

Class grades

Write a program that a teacher can use to compute the average grade for a test. Each time through the loop, the teacher enters a single grade.

When the teacher enters -1, you quit the loop, print the average and end the program. 

sum = 0
count = 0
while True:
    grade = int(input("Enter a grade? "))
    if grade == -1:
        break
    sum += grade
    count += 1

print("Average: ", sum/count)

Best student

Can you add code to the previous program to show the highest score? 

sum = 0
count = 0
max = 0
while True:
    grade = int(input("Enter a grade? "))
    if grade == -1:
        break
    if grade > max:
        max = grade
    sum += grade
    count += 1

print("Average: ", sum/count)
print("Highest: ", max)

n = int(input("Enter a number? "))
count = 1
while count <= 10:

    count += 1

Simulation

Write a program that simulates tossing a coin a million times. Print the total number of heads and tails. 

import random
heads = 0
count = 0
while count < 1000000:
    coin = random.randint(0,1)
    if coin:
        heads += 1
    count += 1

print(heads, "heads, and", 1000000-heads, "tails")

Higher/Lower

Write the following game:

  • The computer chooses a random number (Use random.randint )
  • Ask the user for a number
  • Print a hint voor: "Higher!" or "Lower!"
  • Repeat this until the user guesses the number
import random

# secret number
secret = random.randint(0,100)

x = int(input("your guess?"))

while x != secret:
    if x < secret:
        print("higher!")
    else:
        print("lower!")
    x = int(input("your guess?"))

print("You guessed it!")

Rectangles

Write a program that:

  • Asks the user for a length and a width
  • Prints a rectangle with those dimensions
l = int(input("length?  "))
w = int(input("width? "))

count_l = 0
while count_l < l:
    line = ""
    count_w = 0
    while count_w < w:
        line += "*"
        count_w += 1
    print(line)
    count_l += 1

But you don't actually need a loop: 

l = int(input("length?  "))
w = int(input("width? "))

print((("*"*w) + "\n")*l)
