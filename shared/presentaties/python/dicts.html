Dictionaries

Dictionaries

Store associations: key -> value.

Also called:

  • associative array
  • (hash)map

Example

  • Phonebook: we map names (keys) to phone numbers (values):
d = { "Jenny": '867-5309', "Morgan": '776-2323' }
  • This dictionary has 2 elements
  • keys and values can have (almost) any type (More Info)

Length

A dictionary has a length just like a string and a list 

d = { "Jenny": '867-5309', "Morgan": '776-2323' }
print(len(d))

Output: 

2

Operations

Like with lists, we can add, remove and change values.

Retrieving a value

You retrieve a value by using the key. 

d = { "Jenny": '867-5309', "Morgan": '776-2323' }
print(d["Jenny"])
  • Note the similarity with retrieving an item from a list
  • Dict values are retrieved by key, not by index
  • You cannot retrieve keys this way!

Add/change an element

inventory = {'eggs': 5, 'bread': 8}
inventory['wine'] = 10     # key does not exist: adds new
inventory['bread'] -= 1    # changing existing element
print(inventory)

Output: 

{'eggs': 5, 'bread': 7, 'wine': 10}

Missing keys

If you try to use a non-existing key this gives an error: 

>>> d['marie']
Traceback (most recent call last):
  File "stdin", line 1, in module
KeyError: 'marie'

Check first

d = { "Jenny": '867-5309', "Morgan": '776-2323' }
  if "marie" in d:
      print("Phonenr for marie:", d["marie"])
  else:
      print("Marie is not in phonebook")
  • The in operator works on keys not values!

Another solution

d = { "Jenny": '867-5309', "Morgan": '776-2323' }
  try:
      print("Phonenr for marie:", d["marie"])
  except:
      print("Marie is not in phonebook")

Alternative: get

d = { "Jenny": '867-5309', "Morgan": '776-2323' }
print(d.get("Jenny"))
print(d.get("Marie"))  # Key does not exist -> None
print(d.get("Marie", "unknown contact")) # Default value

Output:
867-5309
None
unknown contact

Looping over a dictionary

for person in d:
    print("Number for", person, "is: ", d[person])

Output: 

Number for Jenny is: 867-5309
Number for Morgan is: 776-2323
  • We loop over the keys
  • Inside the loop we retrieve the values

values()

capitals = { 'DE': 'Berlin', 'FR': 'Paris', 'NL': 'Amsterdam' }
print(capitals.values())

Output: 

['Berlin', 'Paris', 'Amsterdam']
  • You can use this to loop over all values
  • There is a function keys() which works similarly

items()

When you need to retrieve keys and dictionaries at the same time. 

for k, v in capitals.items():
    print(v, "is the capital of", k)

Berlin is the capital of DE
Paris is the capital of FR
Amsterdam is the capital of NL
