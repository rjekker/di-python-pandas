Car

Given the following code: 

class Car:
    def __init__():
        # ... your code here

    def drive(self, distance):
        self.kilometers += distance
        print(f"Driving: {distance}")

    def __str__(self):
        return f"This car has driven {self.kilometers} kilometers"

if __name__=="__main__":
    car = Car()
    car.drive(10)
    car.drive(350)
    print(car)

Note how the drive function uses a property self.kilometers. Finish the __init__ function to make sure that this property exists. Set it to zero.

Check that the program works now. Expected output: 

Driving: 10
Driving: 350
This car has driven 360 kilometers

Cisco Interfaces

(note: if configuring cisco routers is not something you do, you can skip this exercise)

Create a class Interface. It has the following attributes:

  • name: A string with the interface name/number, for example "GigaBitEthernet0/5"
  • ip_address: A string with the ip address for this interface
  • up: A bool, True if this interface should be up, False otherwise)

All of these attributes should be parameters of the __init__ function.

Printing the interface

Add a function __str__ to make it possible to call print for an interface object. This function should return a string containing the commands to configure the interface.

If you run the following code: 

i1 = Interface("GigaBitEthernet3/8", "192.168.12.2", True)
i2 = Interface("FastEthernet4", "192.168.23.18", False)
print(i1)
print(i2)

The output should be as follows: 

interface GigaBitEthernet3/8
    ip 192.168.12.2
    no shutdown
interface FastEthernet4
    ip 192.168.23.18

Rectangle

Create a class Rectangle. This has two properties: a height and a width. Both are arguments of __init()__.

Add a function area() that returns the area of the rectangle.

Print a rectangle

Add a __str__ function that makes it possible to print a rectangle object. This should also print the area.

Note: the __str__ function should return a string, not print it!

Example: 

print(Rectangle(12,54))

Output:
Rectangle: 54 x 12; area: 648

Package

Create a package called geometry; inside it, create a module called rectangle.py. Move your Rectangle class into that module.

Outside of the package, create a new file that imports your new module: 

# This file is outside of the geometry package
from geometry import rectangle

Add a line that creates a Rectangle object and prints it.

List of rectangles

l = [Rectangle(5,8), Rectangle(9,7), Rectangle(23,235), Rectangle(1235,36454)]

Run this code. Does it do what you expect?

Pizza

Consider the following code, which should be placed in toppings.py: 

# in file: topping.py
class Topping:
    def __init__(self, name, price):
        self.name = name
        self.price = price

    def __str__(self):
        return self.name

Start by creating a number of Topping objects, let's say Mozzarella for $0.20, Tomato Sauce for $0.10, Prosciutto for $0.50.

Then, create a Pizza class in a separate file pizza.py. Here are the requirements:

  • A pizza is created with a name and a list of toppings (Margherita has Mozzarella and Tomato Sauce)
  • There should be a function Pizza.price() that returns the price for a pizza. The calculation for this is $2 + the sum of all topping prices.
  • Make it possible to call print() on a Pizza object.

Create several Pizza objects to test your code.

Order

Create a third class, Order, in its own file, order.py. It contains a list of pizzas and an address.

It also has a price() function. This is the total price of all pizzas to deliver; but if that is below $10 you add $3 delivery costs.

Bonus: add an optional argument delivery_notes to the __init__ function. This is a text for the delivery driver, but you can create a pizza without specifying one.

Student grades

Create a class Student. Every student has a name, a list of grades, and the following functions:

  • __init__(self, name) creates student with given name and an empty list of grades
  • add_grade(self, cijfer) add a grade to the list for this student
  • average(self) return the average grade for this student

Note: this exercise is based on an earlier exercise so feel free to look back at the code you wrote before.

Ask the user for a student's name and create a corresponding Student object.

Then, create a loop asking for grades. With every new grade entered, you call add_grade() on the student object and print the new average.

Example 

Student name? Piet
Grade? 5
Average is 5
Grade? 6
Average is 5.5
Grade? 10
Average is 7
