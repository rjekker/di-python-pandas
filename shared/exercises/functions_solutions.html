Statistics: range

See Largest.

Write a function max(a, b, c) that returns the largest of its three arguments.

Write another function min(a, b, c) that returns the smallest of its three arguments.

Write a function range(a, b, c) that returns the range: the difference between the maximum and minimum of the three numbers. 

def max(a, b, c):
    if a > b:
        if a > c:
            return a
        else:
            return c
    else:
        if b > c:
            return b
        else:
            return c

def min(a, b, c):
    if a < b:
        if a < c:
            return a
        else:
            return c
    else:
        if b < c:
            return b
        else:
            return c

def range(a, b, c):
    return (max(a, b, c) - min(a, b, c))

print(range(0, 0, 10))
print(range(100, 40, 1))
print(range(33, 1000, 100))

Define a second function called by_three that takes an argument called number. Uf that number is divisible by 3, by_three should call cube(number) and return its result. Otherwise, by_three should return False. 

def cube(number):
    return number**3

def by_three(number):
    if number % 3 == 0:
        return cube(number)
    else:
        return False

URL

Write a function generate_url(host):

If called with just a hostname, it will return a .nl url. generate_url("rjekker") returns http://www.rjekker.nl.

The function takes several optional arguments:

  • "tld": Set the top-level domain (.nl, .be, .com, etc.)
  • "secure": if True this will create a https url
  • "port": this will set the port number (see below)
  • "path": this will set the path of the url (part after the host/port

Example: 

def generate_url(host, tld="com", secure=False, port=80, path=""):
    if secure:
        protocol = "https://"
    else:
        protocol = "http://"
    if port == 80:
        port_str = ""
    else:
        port_str = ":" + str(port)

    url = protocol + "www." + host + "." + tld + port_str
    if path:
        url += "/" + path
    return url

print(generate_url("pluralsight"))
print(generate_url("codesensei", tld="nl"))
print(generate_url('google', secure=True, port=4444, path="index.html", tld="fr"))
