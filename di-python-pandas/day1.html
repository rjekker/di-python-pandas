Lets's start writing our first pieces of Python code. We learn about basic syntax, types. values and variables as well as two ways to control the flow of execution: if statements and while loops.

Learning goals:

  • Working with variables and basic data types (string, int, float, bool)
  • Making decisions (if)
  • Repeating yourself (if)
  • Writing and running scripts
  • Reading and writing files

Practical applications:

Write a script that reads a file and filters its contents (e.g. find a certain word).
